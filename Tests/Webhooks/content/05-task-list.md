# Content-Launch Checkliste

Bevor ein Blog-Post live geht:

## Inhalt

- [x] Titel ist aussagekräftig und SEO-optimiert
- [x] Einleitung macht neugierig
- [ ] Mindestens ein Bild eingefügt
- [x] Call-to-Action am Ende

## Technik

- [x] Meta-Description gesetzt
- [ ] Open Graph Tags geprüft
- [x] Interne Verlinkung vorhanden
- [ ] Mobile Vorschau getestet

## Distribution

- [ ] Newsletter vorbereitet
- [ ] Social Media Posts geplant
- [x] Webhook-Endpoint konfiguriert
